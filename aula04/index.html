Web Storage JS

Cookies

Salva valores no navegador do usuário como: última vez que acessou o site, última página acessada, etc.

Sessions

Realizam a mesma função do cookie, porém são temporárias, ou seja, quando fechar o navegador, os dados da sessão são perdidos.

Web Storage

No javascript, o webstorage pode ser salvo de duas maneiras diferentes.

localStorage

Funciona quase como um cookie, armazenando localmente no navegador do usuário, mesmo ao fechá-lo.

sessionStorage

Funciona exatamente como uma session, salvando dados enquanto o navegador está aberto.
